# ChiriProject - GitHub Pages

アプリ紹介用のランディングページです。

## ファイル構成

| ファイル | 内容 |
|----------|------|
| `index.html` | トップページ（アプリ一覧） |
| `yesno.html` | YesNo Tonight の紹介ページ |

## セットアップ

1. この `githubpages` フォルダの中身をリポジトリの `docs` にコピー、またはリポジトリルートに配置
2. GitHub の Settings → Pages で Source を設定
   - **Branch**: `main`
   - **Folder**: `/docs` または `/ (root)`

## テーマカラー

- **yesno.html**: yes-no-app の `app/mobile/constants/colors.ts` のカラーパレットを使用

## カスタマイズ

- **App Store リンク**: `yesno.html` の「App Store でダウンロード」ボタンの `href` を、公開後の URL に変更してください。
