# ChiriProject

アプリ紹介用の GitHub Pages サイトです。

## GitHub Pages の設定

- **Settings** → **Pages** → **Build and deployment**
- **Source**: Deploy from a branch
- **Branch**: main
- **Folder**: **`/ (root)`** ← プロジェクトルートから公開

## ファイル

| ファイル | 内容 |
|----------|------|
| `index.html` | トップページ |
| `yesno.html` | YesNo Tonight の紹介 |
